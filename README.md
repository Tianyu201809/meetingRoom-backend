### Meeting Room项目的后端服务

1. 打开项目目录，首先执行 npm install 安装所有依赖
2. 在此项目目录中执行命令行 npm run dev 运行服务器
3. 当显示终端中打印了 “服务器已经启动，端口号3000 连接MongoDB成功” 说明服务器已经启动成功


### 说明
该后端项目目前连接的是我的Mongodb云数据库，如果需要修改的话，需要在config文件夹下找到mongodbConfig文件，在该文件
中配置自己的远程数据库
